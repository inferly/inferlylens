## Datasets

This folder gathers the datasets that are used in test and documentation.

 * `banana.parquet` is a dataset comonly used to illustrate Gaussian process classifications. Basic features are
    * two inputs `['x1', 'x2']`
    * one output `['y']`
    * 5300 entries
 * `airfoil.parquet` is a Kaggle dataset (see the [data card](https://www.kaggle.com/datasets/fedesoriano/airfoil-selfnoise-dataset) for details). Basic features are
    * five inputs `['Frequency','Angle of attack','Chord length','Free-stream velocity', 'Displacement thickness']`
    * one output `['Sound pressure']`
    * 1503 entries
